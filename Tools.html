Calculate slope

This tool is used to calculate slope ratio, PointA below is the first level, PointB is the second level and Distance indicates the distance between two points.

    Calculation Method
  1. A = Subtract PointA and PointB
  2. B = Divide distance with A
  3. Resulting value indicates slope ratio

Please maintain same units for all inputs




Calculate level of next point

This tool is used to calculate the level of next point, PointA below is the first level, Distance indicates the distance between two points and Slope indicates the slope ratio

    Calculation Method
  1. A = Divide distance with slope
  2. B = Add (If location of point is Up) or subtract (If location of point is Down) A with PointA
  3. Resulting value gives value of second point
  4. D =Subtract PointA from B, will give difference between two points

Please maintain same units for all inputs

Calculate distance between two points

This tool is used to calculate the distance between two points/ levels, PointA below is the first level, PointB is the second level Slope indicates the slope ratio

    Calculation Method
  1. A = Subtract PointA from PointB
  2. B = Multiply the result with slope
  3. Resulting value gives distance
